# 🐍 Cobrinha

Uma versão do clássico jogo da Cobrinha (Snake), feita para jogar direto no navegador. Controle a cobrinha, coma a comida para crescer e evite bater nas paredes ou em você mesma.

Feito 100% com **HTML, CSS e JavaScript**, sem frameworks e sem backend.

## 🎮 Como jogar

1. Abra o jogo no navegador.
2. Use as setas do teclado (↑ ↓ ← →) para controlar a direção da cobrinha.
3. Coma a comida que aparece na tela para crescer e ganhar pontos.
4. Evite colidir com as paredes ou com o próprio corpo da cobrinha.
5. O jogo termina quando ocorre uma colisão — tente bater seu recorde!

## 🚀 Como jogar online / rodar localmente

Basta abrir o arquivo `index.html` no navegador — não é necessário instalar nada.

Para rodar localmente:

```bash
git clone https://github.com/lucasabel783-prog/Cobrinha.git
cd Cobrinha
```

Depois é só abrir o `index.html` no navegador de sua preferência.

## 🛠️ Tecnologias usadas

- **HTML5** — estrutura da página
- **CSS3** — estilização e layout
- **JavaScript** — lógica do jogo (movimento, colisão, pontuação)
